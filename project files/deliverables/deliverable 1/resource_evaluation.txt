4.2 Technical Resources 
Present here the computer resources available for the project. That includes computers, software resources (tools, libraries, etc.) available on these computers
The web server used to host the developingproject is the PHPbuilt in web server provided by wamp which come with a bundle of different tools suchas phpmy admin apache and sql buddy. the hosting service for the database on a remote server is heroku. to manage the database we are using dbforge studio for mysql or phpmyadmin version 4.5.3 used with wamp to run the app to access the database remotely. The average computer to host the working project is an intel i5 or smiliar. 2 GB of ram and uses around 30 mb of storage space. ***The operating system used to operate the program as very portable as long as it has php, apache and a mysql server install installed.  Version and course contrl is handled by a private git repository. The framework used to develop the app is Codeigniter 3.03* along with other user interface frameworks such as 3.03 .JQuery, Bootstrap  4****, Normalize v3.0.2 ** and google fonts. The editors to develop the project are with notepad++ or phpStorm by JetBrains. We are using slask to commuicate, google drive to share files and documents. and github to upload finished documents. bug reproting ongit hub.
